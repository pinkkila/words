
## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```


3. Optional: You need an OpenAI API key to use the GPT version of the game.


This app is build with: 
* Sqlite database
* OpenAI Api
* Expo Router
* Expo DocumentPicker
* Expo FileSystem
* SF-symbols
